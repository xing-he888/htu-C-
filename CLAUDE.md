# 题库答题助手 — HTU C++ 考试专用

## 概述

这是一个 **Python + Tkinter** 桌面应用，通过屏幕圈选 → OCR 识别 → 模糊搜索题库 → 弹出答案，帮助答题。

核心能力：
- 圈选屏幕任意位置 → 截图
- PaddleOCR 离线中文识别
- rapidfuzz 模糊匹配搜索答案（对 OCR 错别字有容忍度）
- 多题库源并行搜索

## 常见 C++ 易错点（HTU考试高频考点）

1. **switch 语句** — 没有 break 会穿透（fall-through），常用来出陷阱题
2. **自增/自减** — `x++` vs `++x`，`(x+y)++` 非法（表达式不是左值）
3. **默认参数** — 必须从右往左设默认值，`void f(int x=0, int y)` 是错误的
4. **引用** — 引用定义必须初始化，`int &z;` 错误；不能绑定到临时值
5. **访问控制** — 基类 private 成员在派生类中不可直接访问，但会继承占用内存
6. **构造函数调用顺序** — 先基类后派生类，先成员对象后自身
7. **虚函数机制** — 只有通过基类指针/引用调用时才有多态，值传递会切片
8. **指针运算** — `char *ptr = a;` 指向数组首元素，`*ptr` 解引用
9. **逻辑运算符** — `'A' <= ch <= 'Z'` 语法没错但语义错误，应写 `ch>='A' && ch<='Z'`
10. **数组下标** — 从 0 开始，n 元素数组最大下标 n-1

## 如何使用本工具答题

1. **启动**：运行 `python main.py` 或双击打包好的 `AnswerTool.exe`
2. **确保题库已加载**：主界面显示「C++上机考试题库」已加载 + 题目数量
3. **圈题**：点击「开始圈题」（或 Ctrl+Shift+Q），鼠标拖拽框选屏幕上的题目文字
4. **查看答案**：OCR 识别后自动弹出答案窗口，显示匹配结果（按匹配度排序）
5. **复制**：点击「复制答案」即可粘贴到考试系统

### 圈题技巧

- 只圈选题目正文和选项，不要圈入无关的 UI 元素
- 如果识别效果不佳，可以分多次圈选（先圈题干，再圈选项）
- 程序填空/改错题建议圈入完整代码块

### 模糊搜索说明

- 匹配度 >= 35% 视为有效结果
- OCR 识别出的错别字不影响搜索，算法会做模糊匹配
- 如果搜索无结果，尝试重新圈选更大/更小的区域

## 环境注意事项

- **首次运行需要联网**下载 PaddleOCR 模型（~100MB），之后完全离线可用
- RAR 解压需要 `UnRAR.exe` 放在程序同目录
- 打包命令：`build.bat` → 生成 `dist/AnswerTool/`，整体复制到 U 盘即可无环境运行
- 运行环境：Python 3.8+，依赖见 `requirements.txt`
